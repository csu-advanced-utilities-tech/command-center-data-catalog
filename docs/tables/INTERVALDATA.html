INTERVALDATA

Interval Data (Load Profile)

Meter readings at fixed intervals (e.g., every 15 minutes) used for billing and load profile (LP). Very large and fast-changing table - slow to query.

Grain: One row per endpoint per channel per interval start

Domain: Interval & Register Reads

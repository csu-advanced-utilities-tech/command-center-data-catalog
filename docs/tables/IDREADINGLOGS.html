IDREADINGLOGS

Daily Reading Log

Daily (self) reads received from meters; each data definition is a different read type (kWh, kVAR, etc.). Very large and fast-changing table - slow to query.

Grain: One row per endpoint per data definition per reading date

Domain: Interval & Register Reads
